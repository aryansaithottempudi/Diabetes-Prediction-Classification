# Diabetes Prediction Using Classification Models

## Overview

This project predicts whether a person has diabetes using machine learning classification algorithms.

## Dataset

Kaggle Dataset:
https://www.kaggle.com/datasets/iammustafatz/diabetes-prediction-dataset

Dataset File:
diabetes_prediction_dataset.csv

## Models Used

- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier
- K-Nearest Neighbors (KNN)

## Train/Test Split

- Training Data: 80%
- Testing Data: 20%

## Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1-score
- ROC-AUC
- Confusion Matrix

## Files

- Diabetes_Classification_Final_Submission.ipynb
- diabetes_prediction_dataset.csv (optional)
